## Image Guidelines

1. **Format**: Use JPG for photos, PNG for graphics with transparency
2. **Quality**: Optimize images for web (compress without losing quality)
3. **Aspect Ratio**: Maintain consistent aspect ratios for similar image types
4. **File Size**: Keep images under 500KB for better loading performance
5. **Alt Text**: Always provide meaningful alt text for accessibility

## Placeholder Images

Until you add the actual images, the website will show placeholder backgrounds. The hero section will still function with the gradient background as a fallback.
