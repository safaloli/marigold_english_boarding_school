# Documents Directory

This directory contains downloadable documents for Gyan Jyoti School.

## Required Files

### School Prospectus
- **File**: `school-prospectus.pdf`
- **Description**: Comprehensive school prospectus with admission information, curriculum details, facilities, and school policies
- **Size**: Recommended under 10MB for easy download
- **Content**: Should include:
  - School overview and history
  - Vision and mission
  - Academic programs and curriculum
  - Facilities and infrastructure
  - Admission process and requirements
  - Fee structure
  - School policies
  - Contact information

### Other Documents (Optional)
- `admission-form.pdf` - Printable admission application form
- `fee-structure.pdf` - Detailed fee structure and payment schedule
- `school-calendar.pdf` - Academic calendar with important dates
- `parent-handbook.pdf` - Guidelines for parents and guardians

## File Guidelines
- Use PDF format for consistency
- Keep file sizes reasonable for web download
- Ensure documents are up-to-date
- Include school branding and contact information
- Make documents accessible and easy to read
